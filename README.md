## 技术栈
1. Vue-cli的开发方式（最新3.0无配置单页应用）
2. Vue全家桶：Vue、Vue-router、Vuex、Vue-cli、Vue-devtools、Vue-loader
3. 组件化：单Vue文件
4. 模块化：ES2015 Module
5. 第三方模块: vantUI库, axios, NProgress
6. [postcss适配方案](https://www.w3cplus.com/mobile/vw-layout-in-vue.html)
